HTML: A Estrutura da Web

O que é?

[span_0](start_span)

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar e estruturar o conteúdo de páginas web.[span_0](end_span)

    [span_1](start_span)
  • Natureza Estática: O conteúdo de um arquivo HTML não muda por si só.[span_1](end_span)
    • [span_2](start_span)
  • Conceito Chave: O Hyperlink, que permite a navegação entre páginas.[span_2](end_span)
    • [span_3](start_span)
  • Inventor: Criado pelo físico britânico Tim Berners-Lee.[span_3](end_span)

Características da Sintaxe

    [span_4](start_span)
  • Tags: A sintaxe é baseada em "tags" (etiquetas) que são apresentadas entre os caracteres < e >.[span_4](end_span)
    • [span_5](start_span)
  • Não é Case Sensitive: Não faz distinção entre maiúsculas e minúsculas (<p> é o mesmo que <P>).[span_5](end_span)
    • [span_6](start_span)
  • Flexível: O navegador tenta renderizar a página mesmo com pequenos erros de sintaxe.[span_6](end_span)

Estrutura Básica de um Documento HTML

<!DOCTYPE html>

Define o tipo do documento como HTML5.

<html>

Elemento raiz que define o início da página web.

<head>...</head> [span_7](start_span)

Contém metadados, o título da aba e links para CSS e JavaScript.[span_7](end_span)

<body>...</body> [span_8](start_span)

Área onde todo o conteúdo visível da página é colocado.[span_8](end_span)

</html>

A Evolução: Chega o HTML5

[span_9](start_span)[span_10](start_span)

Lançado oficialmente em 2014, o HTML5 trouxe funcionalidades que revolucionaram o desenvolvimento web, diminuindo a necessidade de plugins externos.[span_9](end_span)[span_10](end_span)

Multimídia e Gráficos Nativos

    [span_11](start_span)
  • <audio>: Para incorporar áudios.[span_11](end_span)
    • [span_12](start_span)
  • <video>: Para incorporar vídeos.[span_12](end_span)
    • [span_13](start_span)
  • <canvas>: Para desenhar gráficos e animações via JavaScript.[span_13](end_span)
    • [span_14](start_span)
  • <svg>: Para gráficos vetoriais escaláveis.[span_14](end_span)

Semântica e Estrutura

[span_15](start_span)

O HTML5 introduziu tags que descrevem o significado do seu conteúdo, melhorando a acessibilidade e o SEO.[span_15](end_span)

    [span_16](start_span)
  • <header>: Cabeçalho.[span_16](end_span)
    • [span_17](start_span)
  • <footer>: Rodapé.[span_17](end_span)
    • [span_18](start_span)
  • <nav>: Links de navegação.[span_18](end_span)
    • [span_19](start_span)
  • <article>, <section>, <aside>: Para estruturar o conteúdo principal.[span_19](end_span)

CSS: O Estilo da Página

O que é?

[span_20](start_span)

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento HTML.[span_20](end_span)

[span_21](start_span)

Ele controla cores, fontes, espaçamentos, layout e como os elementos são mostrados em diferentes mídias (tela, papel, etc.).[span_21](end_span)

Como incluir no HTML?

    [span_22](start_span)
  • Tag <link> (Externo): A forma mais recomendada, ligando a um arquivo .css.[span_22](end_span)
    • [span_23](start_span)
  • Tag <style> (Interno): Dentro do <head>, para estilos específicos daquela página.[span_23](end_span)
    • [span_24](start_span)
  • Atributo style (Inline): Diretamente na tag HTML (não recomendado).[span_24](end_span)

Anatomia de uma Regra CSS

Uma regra CSS é composta por um seletor e um bloco de declaração.

seletor { propriedade: valor; propriedade: valor; }
    [span_25](start_span)
  • Seletor: Aponta para o elemento HTML que você quer estilizar (ex: p, .minha-classe, #meu-id).[span_25](end_span)
    • [span_26](start_span)
  • Declaração: Contém uma ou mais propriedades e valores, separados por ponto e vírgula.[span_26](end_span)

JavaScript: A Interatividade

O que é?

[span_27](start_span)

JavaScript é a linguagem de programação de alto nível usada para criar interatividade em páginas web.[span_27](end_span)

Enquanto HTML é a estrutura e CSS é o estilo, o JavaScript é o comportamento. [span_28](start_span)Ele permite que a página responda às ações do usuário sem precisar recarregar.[span_28](end_span)

O que ele pode fazer?

    [span_29](start_span)
  • Executar algoritmos do lado do cliente (no navegador).[span_29](end_span)
    • [span_30](start_span)
  • Acessar e manipular os elementos da página (o DOM).[span_30](end_span)
    • [span_31](start_span)
  • Interagir com APIs do navegador, como GPS e armazenamento local.[span_31](end_span)
    • [span_32](start_span)
  • Criar animações, validar formulários e buscar dados de um servidor.[span_32](end_span)

O DOM: Conectando Tudo

Document Object Model

[span_33](start_span)

Para que o JavaScript possa manipular uma página, o navegador cria uma representação do documento HTML chamada DOM (Document Object Model).[span_33](end_span)

[span_34](start_span)

O DOM organiza todos os elementos da página em uma estrutura de árvore (DOM Tree).[span_34](end_span)

Como o JavaScript o utiliza?

[span_35](start_span)

O JavaScript usa a API do DOM para navegar e modificar essa árvore.[span_35](end_span)

[span_36](start_span)

É possível encontrar elementos, alterar seus estilos, mudar seus conteúdos ou criar novos elementos, tornando a página dinâmica e interativa.[span_36](end_span)

Formulários: Coletando Dados

[span_37](start_span)Formulários são interfaces que agrupam campos para permitir que o usuário envie informações para um servidor.[span_37](end_span) Eles são a principal forma de interação de entrada do usuário na web.

Principais Elementos de um Formulário:

    [span_38](start_span)
  • Text Box: Para entrada de uma única linha de texto.[span_38](end_span)
    • [span_39](start_span)
  • Text Area: Para múltiplas linhas de texto.[span_39](end_span)
    • [span_40](start_span)
  • Button: Botões clicáveis para enviar ou limpar o formulário.[span_40](end_span)
    • [span_41](start_span)
  • Radio Button: Permite a seleção de apenas uma opção de um grupo.[span_41](end_span)
    • [span_42](start_span)
  • Check Box: Permite a seleção de uma ou mais opções.[span_42](end_span)
    • [span_43](start_span)
  • File "Box": Permite ao usuário selecionar um arquivo local para envio.[span_43](end_span)
