HTML: A Estrutura da Web

O que é?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar e estruturar o conteúdo de páginas web.

  • Inventor: Criado por Tim Berners-Lee no CERN.
  • Conceito Chave: O Hyperlink, que permite a navegação entre documentos.
  • Natureza Estática: O conteúdo de um arquivo HTML, por si só, não muda dinamicamente.

Estrutura Básica

<!DOCTYPE html> <html> <head> <title>Título</title> </head> <body> Conteúdo visível... </body> </html>

HTML5: Tags e Semântica

O HTML5 introduziu elementos semânticos que descrevem claramente seu propósito tanto para o navegador quanto para o desenvolvedor, melhorando a acessibilidade e o SEO.

Estrutura da Página

  • <header>: Define o cabeçalho da página ou de uma seção.
  • <nav>: Agrupa os links de navegação.
  • <main>: Especifica o conteúdo principal e único do documento.
  • <footer>: Define o rodapé da página ou de uma seção.

Agrupamento de Conteúdo

  • <article>: Para conteúdo independente e autossuficiente (ex: post de blog, notícia).
  • <section>: Uma seção genérica de um documento, geralmente com um título.
  • <aside>: Conteúdo lateral, relacionado ao conteúdo principal (ex: sidebar).
  • <figure> e <figcaption>: Para agrupar uma mídia (imagem, vídeo) com sua legenda.

HTML5: Multimídia e Gráficos

Uma das maiores evoluções do HTML5 foi o suporte nativo a elementos de mídia e gráficos, eliminando a dependência de plugins como o Flash.

Áudio e Vídeo

Incorporar mídias se tornou trivial:

<video controls> <source src="filme.mp4" type="video/mp4"> </video> <audio controls> <source src="musica.ogg" type="audio/ogg"> </audio>

Canvas

A tag <canvas> é uma "tela em branco" que permite desenhar gráficos, animações e processar imagens em tempo real usando JavaScript. É a base para jogos e visualizações de dados interativas na web.

SVG: Gráficos Vetoriais para a Web

O que é SVG?

SVG (Scalable Vector Graphics) é um formato de imagem vetorial baseado em XML. Diferente de imagens raster (JPG, PNG), que são baseadas em pixels, os SVGs são definidos por formas matemáticas.

Vantagens

  • Escalabilidade Perfeita: Amplie ou reduza sem perder qualidade.
  • Leveza: Arquivos geralmente menores que os de formatos raster.
  • Editável e Animável: Pode ser modificado com CSS e JavaScript.

Exemplo de Uso

Este código desenha um círculo amarelo com uma borda verde:

<svg width="100" height="100"> <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" /> </svg>

CSS: O Estilo da Página

O que é?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento HTML. Ele separa o conteúdo da sua aparência visual.

Anatomia de uma Regra

seletor { propriedade: valor; }

Como incluir no HTML?

  • Tag <link> (Externo): A forma mais comum e recomendada.
  • Tag <style> (Interno): Dentro do <head> da página.
  • Atributo style (Inline): Diretamente na tag HTML (usar com moderação).

JavaScript & DOM: A Interatividade

JavaScript

É a linguagem de programação que permite implementar itens complexos e interativos em páginas web. É o que torna a web dinâmica.

O que ele pode fazer?

  • Alterar o HTML e o CSS da página em resposta a ações do usuário.
  • Criar animações, validar formulários e buscar dados de um servidor (AJAX).
  • Controlar mídias, desenhar no canvas e muito mais.

DOM (Document Object Model)

Para que o JavaScript possa interagir com a página, o navegador cria uma representação do documento chamada DOM.

O DOM é uma estrutura em árvore onde cada nó é um objeto que representa uma parte do documento. O JavaScript pode manipular esses objetos para alterar dinamicamente a página.

Exemplos Práticos: Introdução

Agora, vamos aplicar os conceitos vistos. Nos próximos slides, construiremos três componentes de interface comuns usando apenas HTML5 semântico e CSS moderno, focando em boas práticas e design responsivo.

Vamos construir:

  • 1. Um Cartão de Perfil Responsivo.
  • 2. Uma Barra de Navegação com Efeito Hover.
  • 3. Um Formulário de Contato Interativo.
9 / 14

Exemplo 1: Cartão de Perfil

HTML Semântico

Usamos <figure> para a imagem e a legenda, e <section> para o conteúdo principal do cartão.

<article class="profile-card"> <figure> <img src="foto.jpg" alt="Foto de Perfil"> </figure> <section class="card-content"> <h2>Nome do Usuário</h2> <p>Desenvolvedor(a) Web</p> </section> </article>

CSS Essencial

Usamos box-shadow para profundidade e border-radius para cantos arredondados.

.profile-card { width: 300px; background: #fff; border-radius: 10px; box-shadow: 0 4px 15px rgba(0,0,0,0.1); overflow: hidden; /* Garante que a imagem não vaze */ transition: transform 0.3s ease; } .profile-card:hover { transform: translateY(-5px); /* Efeito de elevação */ } .profile-card img { width: 100%; display: block; } .card-content { padding: 20px; }

Exemplo 2: Barra de Navegação

HTML Estruturado

A tag <nav> indica que este é um bloco de navegação. Usamos uma lista <ul> para os links, uma prática comum e semântica.

<nav class="main-nav"> <ul> <li><a href="#">Início</a></li> <li><a href="#">Sobre</a></li> <li><a href="#">Projetos</a></li> <li><a href="#">Contato</a></li> </ul> </nav>

CSS com Flexbox

Flexbox simplifica a criação de layouts. Aqui, ele alinha os itens da lista horizontalmente.

.main-nav ul { list-style: none; padding: 0; margin: 0; display: flex; /* A mágica do Flexbox */ justify-content: center; /* Centraliza os itens */ background: #333; } .main-nav a { display: block; padding: 15px 20px; color: white; text-decoration: none; transition: background-color 0.3s; } .main-nav a:hover { background-color: #555; /* Efeito hover */ }

Exemplo 3: Formulário de Contato

HTML5 com Novos Inputs

Usamos <label> para acessibilidade e novos tipos de input do HTML5 como type="email" para validação automática.

<form class="contact-form"> <label for="name">Nome:</label> <input type="text" id="name" required> <label for="email">Email:</label> <input type="email" id="email" required> <label for="message">Mensagem:</label> <textarea id="message" rows="5" required></textarea> <button type="submit">Enviar</button> </form>

CSS com Foco em UX

Estilizamos os campos e usamos a pseudo-classe :focus para dar um feedback visual ao usuário quando ele interage com um campo.

.contact-form input, .contact-form textarea { width: 100%; padding: 10px; margin-bottom: 15px; border: 1px solid #ccc; border-radius: 5px; box-sizing: border-box; /* Garante que o padding não afete a largura */ } .contact-form input:focus, .contact-form textarea:focus { outline: none; border-color: #1a73e8; /* Destaque no foco */ box-shadow: 0 0 5px rgba(26, 115, 232, 0.5); } .contact-form button { padding: 12px 20px; background-color: #1a73e8; color: white; border: none; border-radius: 5px; cursor: pointer; }

Conclusão e Próximos Passos

Você viu a base da construção da web: HTML para a estrutura, CSS para o estilo e JavaScript para a interatividade. As tecnologias modernas e a abordagem semântica do HTML5 nos permitem criar interfaces ricas, acessíveis e eficientes.

Continue explorando:

  • Frameworks CSS: Explore ferramentas como Bootstrap e Tailwind CSS.
  • Pré-processadores CSS: Aprenda sobre SASS/SCSS para escrever CSS mais poderoso.
  • Frameworks JavaScript: Mergulhe em bibliotecas como React, Vue ou Angular para criar aplicações web complexas.
