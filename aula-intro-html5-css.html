O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação que usamos para criar e organizar o conteúdo em uma página da web.

Pense nele como o esqueleto de um site. Ele define os diferentes blocos de conteúdo, como títulos, parágrafos, imagens e links, dando uma estrutura lógica ao documento.

3 / 21

HTML não é uma Linguagem de Programação

HTML (Marcação)

O HTML é uma linguagem declarativa. Você "marca" o conteúdo para dizer ao navegador o que ele é (um título, uma lista, uma imagem), mas não como ele deve se comportar.

  • ✔️ Define a estrutura e o significado.
  • ❌ Não possui lógica condicional (if/else).
  • ❌ Não possui laços de repetição (loops).
  • ❌ Não pode executar cálculos ou manipular dados.

Linguagem de Programação

Uma linguagem de programação, como JavaScript, é usada para criar lógica e comportamento. Ela diz ao computador como fazer as coisas.

  • ✔️ Executa algoritmos e tarefas.
  • ✔️ Usa lógica, loops e funções.
  • ✔️ Responde a eventos do usuário.
  • ✔️ Manipula dados e se comunica com servidores.

Estrutura Básica de um Arquivo HTML5

<!DOCTYPE html>

Informa ao navegador que o documento é HTML5.

<html lang="pt-BR">

Elemento raiz. O atributo `lang` define o idioma da página.

<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Título da Página</title> </head>

Contém metadados: `charset` para codificação de caracteres, `viewport` para responsividade e `title` para o título da aba.

<body> <!-- Todo o conteúdo visível da página vai aqui --> </body> </html>

Corpo do documento, onde todo o conteúdo visual é inserido.

Organizando Conteúdo: Listas

Listas Não Ordenadas `<ul>`

Usadas quando a ordem dos itens não importa. Geralmente renderizadas com marcadores (bolinhas).

<ul> <li>Café</li> <li>Leite</li> <li>Pão</li> </ul>

Listas Ordenadas `<ol>`

Usadas quando a sequência dos itens é importante. Renderizadas com números ou letras.

<ol> <li>Primeiro, misture os ingredientes.</li> <li>Segundo, asse por 30 minutos.</li> <li>Terceiro, sirva quente.</li> </ol>
6 / 21

Inserindo Imagens `<img>`

A tag `<img>` é usada para incorporar imagens em uma página. Ela é uma tag "vazia", ou seja, não possui fechamento.

<img src="caminho/para/imagem.jpg" alt="Descrição da imagem">

Atributo `src` (Source)

É obrigatório e especifica o caminho (URL) para o arquivo de imagem que será exibido.

Atributo `alt` (Alternative Text)

Fornece um texto alternativo para a imagem. É crucial para a acessibilidade, pois é lido por leitores de tela para usuários com deficiência visual e exibido se a imagem não carregar.

Semântica para Imagens: `<figure>` e `<figcaption>`

Para associar semanticamente uma imagem a uma legenda, o HTML5 oferece as tags `<figure>` e `<figcaption>`.

  • <figure>: "Embrulha" a mídia (imagem, diagrama, etc.) que faz parte do fluxo principal, mas poderia ser movida para outro lugar (como um apêndice) sem afetar o sentido do texto.
  • <figcaption>: Fornece a legenda para o conteúdo dentro do `<figure>`.

Exemplo Prático

<figure> <img src="por-do-sol.jpg" alt="Pôr do sol na praia de Ipanema"> <figcaption>Fig.1 - Pôr do sol visto do Arpoador, Rio de Janeiro.</figcaption> </figure>

Antes do CSS: A Era do Caos

Nos primórdios da web, não havia separação entre estrutura e estilo. A aparência era definida diretamente no HTML, usando tags e atributos que hoje são obsoletos.

Como era feito:

<body bgcolor="#FFFFFF"> <center> <font color="blue" size="5"> Este é um título azul! </font> </center> <p>Um parágrafo simples.</p> </body>

Problemas: Manutenção difícil, código poluído e nenhuma consistência visual entre páginas. Mudar a cor de todos os títulos significava editar cada arquivo HTML individualmente.

9 / 21

CSS: O Estilo da Página

CSS (Cascading Style Sheets) é a linguagem que usamos para estilizar documentos HTML. Pense nele como o maquiador e designer de interiores de um site.

Ele foi criado para resolver o problema da estilização no HTML, permitindo separar completamente a apresentação (cores, fontes, layout) da estrutura (conteúdo).

10 / 21

CSS também não é uma Linguagem de Programação

Assim como o HTML, o CSS é uma linguagem declarativa, não de programação. Você declara "regras" de estilo que o navegador deve aplicar aos elementos.

  • ✔️ Descreve a aparência dos elementos.
  • ✔️ Pode ter "variáveis" (Custom Properties) e cálculos simples (calc()).
  • ❌ Não possui a lógica computacional completa de uma linguagem de programação (loops, condicionais complexas, funções para manipular dados).

O CSS determina como algo deve parecer, enquanto uma linguagem de programação determina como algo deve funcionar.

3 Formas de Incluir CSS

1. Externo (Recomendado)

Um arquivo .css separado é linkado no <head>. Permite reutilizar estilos em várias páginas.

<link rel="stylesheet" href="estilos.css">

2. Interno

O código CSS é colocado dentro da tag <style> no <head>. Útil para estilos específicos de uma única página.

<style> body { background-color: #f0f0f0; } </style>

3. Inline (Evitar)

O estilo é aplicado diretamente no elemento através do atributo `style`. Dificulta a manutenção.

<p style="color: blue; font-size: 18px;">...</p>

CSS: O Modelo de Caixa (Box Model)

Todo elemento HTML pode ser considerado uma caixa retangular. O Box Model é o conjunto de regras que define como o tamanho e o espaçamento dessa caixa são calculados.

  • Content: O conteúdo real (texto, imagem).
  • Padding: Espaçamento interno, entre o conteúdo e a borda.
  • Border: A borda que envolve o padding.
  • Margin: Espaçamento externo, fora da borda, que afasta o elemento de outros.

CSS Avançado: Pseudo-classes e Variáveis

Pseudo-classes

São palavras-chave adicionadas a seletores que especificam um estado especial do elemento. Permitem estilizar elementos de forma dinâmica.

/* Estilo quando o mouse está sobre o link */ a:hover { text-decoration: underline; } /* Estilo quando o campo está focado */ input:focus { border-color: blue; } /* Estilo para cada linha ímpar de uma lista */ li:nth-child(odd) { background-color: #eee; }

Variáveis (Custom Properties)

Permitem armazenar valores reutilizáveis, facilitando a manutenção e a criação de temas.

:root { --cor-primaria: #0056b3; --tamanho-fonte: 16px; } body { font-size: var(--tamanho-fonte); } a { color: var(--cor-primaria); }

HTML5: A Era da Multimídia

Com o HTML5, incorporar áudio e vídeo se tornou nativo, sem a necessidade de plugins. As tags <audio> e <video> oferecem controle total sobre a mídia.

Atributos Comuns

  • controls: Exibe os controles padrão do navegador (play, pause, volume).
  • autoplay: Inicia a reprodução automaticamente (geralmente requer `muted`).
  • muted: Inicia a mídia sem som.
  • loop: Faz a mídia reiniciar ao terminar.
  • poster: (para vídeo) Exibe uma imagem antes do vídeo ser carregado.

Uso com a tag `<source>`

Para maior compatibilidade, você pode fornecer múltiplos formatos de arquivo. O navegador usará o primeiro que conseguir reproduzir.

<video controls poster="capa.jpg"> <source src="video.mp4" type="video/mp4"> <source src="video.webm" type="video/webm"> Seu navegador não suporta a tag de vídeo. </video>

JavaScript & DOM: A Interatividade

O que eles fazem juntos?

O JavaScript é a linguagem que dá vida à página. Para fazer isso, ele manipula o DOM (Document Object Model).

O DOM é uma representação em árvore de todo o seu código HTML. O JavaScript pode navegar nessa árvore para encontrar, adicionar, remover ou modificar qualquer elemento, tornando a página dinâmica e interativa.
