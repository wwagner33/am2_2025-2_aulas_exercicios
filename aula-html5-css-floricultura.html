Guia Prático: Construindo a Vitrine da Floricultura

Um tutorial de HTML5 e CSS para estruturar sua primeira página de e-commerce.

Parte 1: Requisitos da Interface (HTML/CSS)

Com base na descrição do projeto, e focando apenas na estrutura visual (layout) e sem a interatividade (JavaScript), os requisitos para esta etapa são:

  1. Definir a estrutura semântica principal da página (Cabeçalho, Conteúdo Principal, Rodapé).
  2. Configurar o cabeçalho (<header>) para conter o título da loja (<h1>) e um link (<a>) para a "Home", utilizando um ícone.
  3. Estruturar a área de conteúdo principal (<main>) para abrigar a listagem de produtos.
  4. Organizar a exibição dos produtos em um layout de grade (Grid).
  5. Criar um componente "card" (<div>) para representar visualmente cada produto.
  6. Exibir a imagem do produto (<img>) dentro de cada card, utilizando a tag <figure> para semântica.
  7. Apresentar o nome do produto (<h2> ou <h4>).
  8. Descrever o produto em um parágrafo (<p>).
  9. Mostrar o preço do produto (<p> ou <h4>).
  10. Incluir um link (<a>) estilizado como ícone para "Adicionar ao Carrinho".
  11. Configurar os metadados essenciais da página (<meta>), definindo o charset para "UTF-8" e o lang para "pt-br".
  12. Vincular (<link>) a folha de estilos CSS externa ao documento HTML.
  13. Importar a fonte "Roboto" e os ícones "Material Symbols" (ou Fonts) do Google Fonts.
  14. Estilizar os elementos globais (tamanho da fonte padrão, cores de fundo e de fonte, espaçamento entre linhas).
  15. Aplicar espaçamentos (padding e margin) adequados, especialmente nas imagens e nos cards.
  16. Definir um rodapé (<footer>) simples com informações da loja (ex: copyright).

Parte 2: Conceitos Essenciais (HTML5 e CSS)

Vamos revisar as ferramentas que usaremos para construir essa interface.

Conceitos de HTML5

O HTML (HyperText Markup Language) é a linguagem de marcação que estrutura o conteúdo da web. O HTML5 introduziu tags "semânticas" que dão significado ao conteúdo.

<!DOCTYPE html>
Define o documento como HTML5.
<html lang="pt-br">
Elemento raiz que envolve todo o conteúdo, com o atributo lang definindo o idioma.
<head>
Contém metadados sobre a página (título, links para CSS, metatags).
<meta charset="UTF-8">
Garante que o navegador exiba corretamente caracteres especiais (acentos, etc.).
<link>
Usada para vincular arquivos externos, como nossa folha de estilos (CSS).
<body>
Contém todo o conteúdo visível da página.
<header>
Define o cabeçalho da página ou de uma seção (geralmente contém o logo, título principal e navegação).
<main>
Define o conteúdo principal e único da página. (Não deve incluir cabeçalhos, rodapés ou barras laterais repetitivas).
<footer>
Define o rodapé da página ou de uma seção (geralmente contém informações de copyright, links de contato, etc.).
<div>
Um "container" genérico usado para agrupar elementos, muito útil para aplicar estilos CSS (como o nosso "card").
<h1>, <h2>, <h4>
Títulos (Headings). Definem a hierarquia da informação. <h1> é o mais importante.
<p>
Define um parágrafo de texto.
<a>
Âncora (link). Usado para criar hiperlinks. O atributo href define o destino do link.
<figure>
Usado para agrupar conteúdo de mídia (como uma imagem) e, opcionalmente, sua legenda (<figcaption>).
<img>
Exibe uma imagem. O atributo src define o caminho (URL) da imagem e o alt fornece um texto alternativo (essencial para acessibilidade).

Para saber mais

A MDN Web Docs (Mozilla Developer Network) é a referência mais completa para desenvolvedores web. Você pode consultar o que cada tag HTML faz e ver exemplos.

  • Referência (HTML): MDN - Referência de Elementos HTML

Conceitos de CSS

O CSS (Cascading Style Sheets) é a linguagem usada para definir a aparência e o layout dos elementos HTML.

@import (Google Fonts)
Usaremos a regra @import dentro do nosso arquivo CSS para carregar a fonte "Roboto" e os ícones do Google.
Seletores (Ex: .card)
Formas de "selecionar" elementos HTML para aplicar estilos. Um seletor de classe, como .card, seleciona todos os elementos com o atributo class="card".
Box Model (margin/padding)
margin é o espaçamento fora da borda (entre elementos), e padding é o espaçamento dentro da borda (entre o conteúdo e a borda).
CSS Grid (display: grid)
Um sistema de layout bidimensional (linhas e colunas). Usaremos display: grid; no container pai para organizar os cards.
grid-template-columns
Define o número e o tamanho das colunas do grid. Usaremos repeat(auto-fit, minmax(250px, 1fr)) para criar um grid responsivo.
Propriedades de Fonte
font-family (tipo da fonte), font-size (tamanho), color (cor do texto) e line-height (espaçamento entre linhas).

Para saber mais

Para aprender CSS Grid de forma visual e interativa, o "Grid Garden" é um excelente recurso. Para referência de propriedades, a W3Schools é rápida e direta.

  • Referência (CSS Grid): CSS-Tricks - A Complete Guide to Grid
  • Referência (Propriedades CSS): W3Schools - CSS Reference

Parte 3: Tutorial Prático (HTML + CSS)

Vamos criar a página principal da floricultura, com a listagem de produtos.

Passo 1: Estrutura de Arquivos

Crie uma pasta para o projeto (ex: floricultura) e, dentro dela, crie dois arquivos:

  1. index.html (Nosso HTML)
  2. style.css (Nosso CSS)

Passo 2: O HTML Base (index.html)

Vamos usar o VSCode para agilizar a criação da estrutura base.

Parada obrigatória: VSCode Emmet

No arquivo index.html vazio, digite ! (ponto de exclamação) e pressione a tecla Tab ou Enter.

O VSCode (usando o recurso "Emmet") criará automaticamente a estrutura HTML básica. Altere o atributo lang de "en" para "pt-br" e o <title> para "Floricultura da Vila".

Adicione os links para o CSS e para os Ícones do Google dentro do <head>:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Floricultura da Vila</title>
    
    <!-- 1. Link para nosso CSS -->
    <link rel="stylesheet" href="style.css">
    
    <!-- 2. Link para os Ícones do Google Fonts -->
    <link href="https://fonts.googleapis.com/icon?family=Material+Icons"
      rel="stylesheet">
</head>
<body>
    
    <!-- O conteúdo virá aqui -->

</body>
</html>

Passo 3: O CSS Base (style.css)

Agora, vamos ao style.css para importar a fonte Roboto e definir estilos globais.

/* 1. Importando a Fonte Roboto */
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

/* Sugestão: Um "Reset" básico de CSS.
   Faz com que 'padding' e 'border' sejam incluídos 
   no tamanho total do elemento, facilitando o layout.
*/
* {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}

/* 2. Estilos Globais (Body) */
body {
    font-family: 'Roboto', sans-serif; /* Fonte Padrão */
    font-size: 16px; /* Tamanho da fonte padrão */
    line-height: 1.6; /* Espaçamento entre linhas */
    background-color: #f9f9f9; /* Cor de fundo suave */
    color: #333; /* Cor da fonte principal */
}

Passo 5: Estilizando o Layout e os Cards (style.css)

Finalmente, vamos fazer a mágica do CSS Grid e estilizar os cards. Adicione este código ao final do seu style.css:


/* --- Estilos do Layout --- */
header {
    background-color: #006400; /* Verde escuro */
    color: #fff;
    padding: 1rem 2rem;
    display: flex; /* Usando flexbox para alinhar o título e o ícone */
    justify-content: space-between;
    align-items: center;
}

header h1 {
    font-size: 2rem;
}

header a {
    color: #fff;
    text-decoration: none;
}

/* Centraliza o conteúdo principal e o rodapé */
main, footer {
    max-width: 1200px;
    margin: 1.5rem auto; /* Centraliza na página */
    padding: 0 1rem;
}

footer {
    text-align: center;
    color: #777;
    border-top: 1px solid #ddd;
    padding-top: 1rem;
}

main h2 {
    margin-bottom: 1.5rem;
    color: #006400;
}

/* --- O Layout GRID --- */
.product-container {
    display: grid;
    /* Aqui definimos as colunas:
       'repeat' - repita o padrão.
       'auto-fit' - crie quantas colunas couberem na tela.
       'minmax(300px, 1fr)' - cada coluna deve ter no MÍNIMO 300px, 
       mas pode crescer (1fr) para preencher o espaço igualmente.
    */
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 1.5rem; /* Espaçamento entre os cards */
}


/* --- Estilos do Card --- */
.product-card {
    background-color: #fff;
    border: 1px solid #ddd;
    border-radius: 8px;
    box-shadow: 0 2px 5px rgba(0,0,0,0.1);
    overflow: hidden; /* Garante que a imagem não "vaze" */
    display: flex;
    flex-direction: column; /* Organiza o card internamente */
}

/* Margin e Padding da Imagem */
.product-card figure {
    /* 'margin: 0' já foi feito no reset global (*) */
}

.product-card img {
    width: 100%; /* Faz a imagem ocupar 100% do 'figure' */
    height: 200px; /* Altura fixa para as imagens */
    object-fit: cover; /* Garante que a imagem cubra a área sem distorcer */
    display: block; /* Remove espaço extra abaixo da imagem */
}

/* Padding do conteúdo do card */
.product-card h4,
.product-card p,
.product-card .cart-button {
    padding-left: 1rem;
    padding-right: 1rem;
}

.product-card h4 {
    color: #333;
    font-size: 1.25rem;
    margin-top: 1rem;
    margin-bottom: 0.5rem;
}

.product-card p {
    font-size: 0.9rem;
    color: #555;
    flex-grow: 1; /* Faz a descrição empurrar o preço para baixo */
    margin-bottom: 1rem;
}

.product-card .price {
    font-size: 1.1rem;
    font-weight: bold;
    color: #006400;
    margin-bottom: 1rem;
    flex-grow: 0; /* Reseta o flex-grow */
}

.cart-button {
    display: flex;
    align-items: center;
    justify-content: center;
    background-color: #ffc400; /* Amarelo */
    color: #333;
    text-decoration: none;
    text-align: center;
    padding: 0.75rem;
    font-weight: bold;
    margin: 0; /* O botão ocupará a largura toda, sem margin interna */
    transition: background-color 0.3s ease;
}

.cart-button .material-icons {
    margin-right: 0.5rem; /* Espaço entre o ícone e o texto */
    font-size: 1.2rem;
}

.cart-button:hover {
    background-color: #e6b100;
}
